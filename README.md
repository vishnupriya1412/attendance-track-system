# 📊 Attendance Tracking System

## 📌 Overview
The Attendance Tracking System is a web-based application designed to efficiently manage and monitor student attendance. It helps institutions maintain accurate records, reduce manual work, and generate reports بسهولة.

---

## 🚀 Features
- User Authentication (Admin/Teacher login)
- Add, update, and delete student details
- Mark daily attendance
- View and manage attendance records
- Generate attendance reports
- Search and filter student data
- Dashboard with attendance summary

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Python (Flask)  
- **Database:** SQLite / MySQL  
- **Libraries:** Pandas, NumPy, OpenCV (optional)

## ⚙️ Installation & Setup

1. Clone the repository:
git clone https://github.com/your-username/attendance-tracker.git
Navigate to project folder:
cd attendance-tracker

Install dependencies:
pip install -r requirements.txt

Run the application:
python app.py

Open in browser:
http://127.0.0.1:5000/
